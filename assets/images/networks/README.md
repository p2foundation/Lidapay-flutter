# Network Logos

Add your network logo files to this directory. The app expects the following files:

- `auto_detect.png` - Auto-detect network icon
- `airteltigo.png` - AirtelTigo logo
- `expresso.png` - EXPRESSO logo
- `glo.png` - GLO logo
- `mtn.png` - MTN logo
- `tigo.png` - TIGO logo
- `telecel.png` - Telecel logo
- `busy.png` - Busy logo
- `surfline.png` - Surfline logo

## Logo Specifications
- Format: PNG (recommended) or JPG
- Size: 48x48 pixels (will be displayed at 24x24)
- Background: Transparent (for PNG)
- Keep logos simple and recognizable at small sizes

The app will automatically fall back to a SIM card icon if any logo is missing.
